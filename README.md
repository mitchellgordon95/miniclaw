# MiniClaw

Minimal personal assistant daemon using Claude Code.

Built for one persistent agent (Wright), multi-channel input (web/SMS), and 30-minute heartbeats.

## Philosophy
- Small, understandable codebase
- Claude Code for everything (no API costs via Max subscription)
- Workspace-centric (all state in one place)
- Easy to tinker with
